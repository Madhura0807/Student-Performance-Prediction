About the Project

The Student Performance Prediction web application combines machine learning and modern web design to estimate math scores from educational and demographic attributes. It showcases a clean MVC-style Flask architecture with modular training and prediction pipelines.

Users can enter a student's profile, receive a predicted score, and see an interpretation of the anticipated performance. The app also includes model comparison visuals and feature importance insights.

Educational Use

Useful for academic planning, performance monitoring, and student support initiatives.
