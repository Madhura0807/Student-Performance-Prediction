# Student Performance Prediction

A complete Flask-based machine learning web application that predicts a student's math score based on academic and demographic inputs.

## Features
- Responsive dashboard-style UI with Bootstrap and modern gradients
- Home, About, Prediction, Result, Model Info, and Contact pages
- Trained regression models including Linear Regression, Decision Tree, Random Forest, XGBoost, and CatBoost
- Automatic best model selection and persistence using dill
- Visualizations for score distribution, model comparison, and feature importance
- Form validation, error handling, logging, and loading animation

## Setup
1. Create and activate a virtual environment
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Train the model and generate artifacts:
   ```bash
   python app.py
   ```
4. Open http://127.0.0.1:5000/

## Project Structure
- app.py
- requirements.txt
- setup.py
- artifacts/
- notebook/
- src/
- templates/
- static/
- logs/
